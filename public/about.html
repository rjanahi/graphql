About this Project

1) Project Overview

This is a minimal GraphQL Profile web app. After a successful login (JWT), it fetches your data from the GraphQL API and renders a profile with three core sections (user info & totals, audit ratio, categorized XP) plus a dedicated statistics area with SVG-based charts.

2) Tech Stack

Frontend
  • Vanilla HTML, CSS, JavaScript
  • SVG for charts (no external chart libs)
  • JWT stored in localStorage
Backend
  • Go (Golang) server (project requirement)
  • GraphQL endpoint: https://learn.reboot01.com/api/graphql-engine/v1/graphql
  • Bearer authentication

3) Features

  • Login with JWT, graceful error handling for invalid credentials
  • Three required profile sections sourced directly from GraphQL
  • SVG charts: XP by Activity (bar), XP Progression (line), and Audit Ratio (bars)
  • Logout clears auth and returns to login
  • Reusable fetchGraphQL() helper for typed queries

4) GraphQL Info

The page primarily runs three queries. You can copy and test them in GraphiQL (uses your stored JWT).

User basics 
query Me {
  user {
    id
    login
    email
    firstName
    lastName
  }
}
Audit ratio 
query Audit {
  user {
    auditRatio
    totalUp
    totalDown
  }
}
XP transactions (filtered) 
query Xp {
  transaction(
    where: { type: { _eq: "xp" }, path: { _like: "/bahrain/bh-module/%" } }
    order_by: { createdAt: asc }
    limit: 1000
  ) {
    amount
    createdAt
    object { name type path }
  }
}
